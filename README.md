## Usage

    source init.sh  # pre-defined alias and functions.  
    export docker_user=YOUR_DOCKER_HUB_USER # for docker_build
    docker_build--folder---tag <folder> [tag]

    ./run.sh    # pre-defined `docker run -d` commands
